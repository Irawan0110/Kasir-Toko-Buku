# Kasir Toko Buku
Sebuah program menggunakan Python yang berguna untuk mengitung total semua pesanan barang yang dibeli pada toko buku.

Program ini masih dalam tahap pengembangan.

# Fitur
* Pilih barang yang dibeli
* Jumlah barang
* Total
* Masukan uang anda
* Kembalian
* Clear
